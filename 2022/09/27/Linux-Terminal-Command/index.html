Linux Terminal Command

Posted on 2022-09-27

필수 리눅스 터미널 명령어

1. 기본

man

manul의 약자로 명령어에 대해서 자세히 알아볼 수 있습니다.

clear

터미널 스크린을 깨끗하게 만들어 줍니다.

2. 파일 탐색

ls

list의 약자로 디렉토리 목록을 보여준다. 현재 경로에 어떤 폴더와 파일이 있는지 확인하고 싶을때 사용합니다.

cd

change directory의 약자로 폴더들을 자유자재로 넘나들 수 있습니다. 즉, 디렉토리간 이동이 가능합니다.

pwd

print working directory의 약자로 절대 경로를 말합니다. 지금 내가 어떤 경로에 있는지, 내가 현재 있는 곳의 모든 경로를 프린트 해줍니다.

open [경로]

해당하는 경로를 Finder에서 열어줍니다.

find

파일 시스템에서 특정 파일 또는 디렉토리를 찾을때 사용합니다. 예를들어 만약 내가 현재 있는 경로와 그 하위에 있는 모든 폴더에 한해서 모든 텍스트 파일을 찾고 싶다면 find . -type file -name "*.txt"를 하면됩니다.

which

프로그램이 설치된 경로를 알 수 있습니다.
